Exercise on HTML, CSS, JS and Path-finding logic.

The idea is to create and host a simple webpage containing:
  * A title element;
  * A rectangular visualizer where the logic will be displayed;
  * Four buttons to swap between each algorithm;
  * The algorithms are:
    * ASTAR;
    * BFS;
    * DFS;
    * DIJKSTRA.


  The skeleton of the page is pure HTML.
  The flourish is CSS.
  The logic is JS.

  All the information and source are open to use and iterate over.
